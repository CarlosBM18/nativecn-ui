## Overview

`Badge` is a React Native component for displaying brief labels or tags with optional variants. It provides customization through its `label` and `variant` properties and automatically adapts to the device's current color scheme.

## Installation

Include the `Badge.tsx` file in your React Native project. [File Link](https://github.com/Mobilecn-UI/nativecn-ui/blob/main/components/Badge.tsx)

## Properties

- `label: string` (Required) - The text displayed inside the badge.
- `variant?: Variant` (Optional) - The variant of the badge. Can be one of 'default', 'secondary', or 'destructive'.

> **Note**: The `variant` property determines the color of the badge. The color scheme (light/dark mode) is automatically detected.

## Usage

### Basic Usage with Variants

Here's how you can use the `Badge` component:

```jsx
<Badge label="New" />
<Badge label="Archived" variant="secondary" />
<Badge label="Error" variant="destructive" />
```
