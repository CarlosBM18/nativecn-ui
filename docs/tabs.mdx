## Overview

`Tabs` is a versatile React Native component designed for creating tab-based user interfaces. This component set comprises `Tabs`, `TabsList`, `TabsTrigger`, and `TabsContent`, which work together to deliver a seamless tabbing experience.

## Components

### Tabs

The parent container for the tabbing system.

- `children`: Components to be rendered inside the Tabs container.

### TabsList

The container for individual tab triggers.

- `children`: `TabsTrigger` components.

### TabsTrigger

Triggers or buttons for each tab.

- `id: number` (Required) - A unique identifier for the tab.
- `title: string` (Required) - The label displayed on the tab trigger.
- `activeTab: number` (Required) - The ID of the currently active tab.
- `setActiveTab: (id: number) => void` (Required) - A function to change the active tab.

### TabsContent

Container for the content of each tab.

- `activeTab: number` (Required) - The ID of the currently active tab.
- `children`: The content of each tab. This should match the number of `TabsTrigger` components.

## Usage

### Basic Usage

Here's how you can use the Tabs component system:

```jsx
const tabs = [
  { id: 0, title: 'Tab 1' },
  { id: 1, title: 'Tab 2' },
];

<Tabs>
  <TabsList>
    {tabs.map(tab => (
      <TabsTrigger
        key={tab.id}
        id={tab.id}
        title={tab.title}
        activeTab={activeTab}
        setActiveTab={setActiveTab}
      />
    ))}
  </TabsList>
  <TabsContent activeTab={activeTab}>
    <View>
      <Text className="text-black dark:text-white">Tab 1 content</Text>
    </View>
    <View>
      <Text className="text-black dark:text-white">Tab 2 content</Text>
    </View>
  </TabsContent>
</Tabs>;
```
