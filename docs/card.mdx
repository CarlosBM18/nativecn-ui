

## Overview

`Card` is a versatile React Native component that provides a structured format for displaying content. The card can display title, description, main content, and footer text. It also comes with automatic adaptation to the device's current color scheme.

## Installation

Include the `Card.tsx` file in your React Native project. [File Link](https://github.com/nativecn-UI/nativecn-playground/blob/main/Nativecn%20Playground.components/Card.tsx)

## Properties

- `title?: string` (Optional) - The title of the card.
- `description?: string` (Optional) - A brief description or subtitle.
- `content?: string` (Optional) - The main content inside the card.
- `footer?: string` (Optional) - Text to display at the bottom of the card.
- `variant?: Variant` (Optional) - The variant of the card. Currently, only the 'default' variant is supported which determines the card's color based on the color scheme.

> **Note**: The `variant` property determines the color of the card. The color scheme (light/dark mode) is automatically detected.

## Usage

### Basic Usage

Here's how you can use the `Card` component:

```jsx
<Card
    title="Accelerate UI"
    description="Enter a new development experience"
    content="Sleek, easy to use components to build your next mobile app faster. Inspired by the legendary shadcn/ui."
/>
```
