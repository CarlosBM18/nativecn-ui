## Why Nativecn UI?

- **TypeScript Integrated**: Static typing for all components.
- **Customizable**: Modify components using Tailwind CSS classes.
- **Effortless Integration**: Incorporate the component files into your project. No intricate prerequisites.
- **Beautifully Designed**: With a focus on aesthetics, each component is carefully crafted to provide a visually appealing experience.

## Installation

#### 1. Create a new typescript expo project

```bash
npx create-expo-app -t expo-template-blank-typescript
```

#### 2. Setup Tailwind CSS

[Nativewind](https://www.nativewind.dev/quick-starts/expo)

#### 3. Add Nativecn UI Components

[Nativecn UI Components](https://github.com/Mobilecn-UI/nativecn-ui)

Simply add the components directory or copy paste the required components.
